# 🍵 Chai

A personal header-only utility library for the C programming language.
Chai is intended to be an easy-to-use tool that provides basic data structures such as lists and strings.

## Types

* Chai_View
* Chai_List
* Chai_List_I32
* Chai_List_F32
* Chai_String
